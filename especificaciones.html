Especificaciones de Adivina!

Caso de uso Adivina!
Objetivo Adivinar, en la menor cantidad de intentos, un número elegido por la computadora al azar.
Precondiciones No hay precondiciones

Curso normal Resultado esperado
El usuario ingresa a la aplicación.
El sistema muestra la pantalla:
  • Títulos:
    • Adivina!
    • Juego en el que la máquina elige un número al azar y el usuario intenta adivinarlo recibiendo algunas pistas.
  • Campos
    • Número (campo de texto que espera recibir un número por parte del usuario y compararlo con el que adivnó la máquina. Valor por defecto: Vacío.
  • Acciones
    • Prueba Suerte!
    • Nuevo Juego
1. El usuario selecciona un número y lo digita en el campo 'Numero'.
2. El usuario selecciona la opción "Prueba Suerte!" o la tecla ENTER.
3.1 El número elegido es menor al seleccionado por la aplicación. El sistema muestra un mensaje indicando si se debe ingresar un número mayor.
3.2 El número elegido es mayor al seleccionado por la aplicación. El sistema muestra un mensaje indicando si se debe ingresar un número menor.
4. El usuario selecciona la opción "Nuevo Juego". El sistema selecciona otro número aleatorio y comienza un nuevo juego.
